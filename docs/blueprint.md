# **App Name**: LuxAI Platform

## Core Features:

- User Authentication & Profile: Enable secure user registration, login, and basic profile management for personalized access to AI studios.
- AI Studios Catalog: A dynamic listing page showcasing various AI studios (e.g., SmartFolio, Music, Blog) with status indicators (Live, Beta, Coming Soon).
- DLX SmartFolio (AI Investment Insights Tool): A dedicated studio interface allowing users to input financial data and receive AI-generated portfolio insights, risk analysis, or investment strategy suggestions as a tool.
- DLX Chat with AI Agent Personas: An interactive chat interface leveraging generative AI. Users can select and interact with different AI agent personas (e.g., Lux, Architect, Dev) to receive specialized assistance, answer queries, and perform tasks as an intelligent agent tool. All agents are linked to Gemini free tier.
- Persistent User Data Storage: Utilize Firebase Firestore for securely storing user accounts, preferences, AI-generated content (e.g., SmartFolio reports), and chat conversation history.
- Responsive Dashboard Navigation: Implement an intuitive and responsive navigation system for the main dashboard, studios, chat, agents, news, labs, and settings areas, ensuring a seamless user experience across devices.
- AI Agent & Persona Management: A dedicated interface to view, configure, and manage various AI agents and their personas, allowing users to define roles, capabilities, and access status (e.g., Live, Beta) for different AI assistants. All agents are linked to Gemini free tier.
- Platform News Feed: A section providing updates, announcements, and relevant news related to the LuxAI Platform, its features, and AI developments.
- Labs Hub (Experimental Projects & Roadmap): An interface to track and manage experimental AI projects, features, and agent assignments, including progress tracking and status indicators (e.g., Active, Concept).
- User Settings & Preferences: Allow users to customize their platform settings, notifications, security preferences, and manage personal data.

## Style Guidelines:

- Primary color: Deep, vibrant indigo (#6626D9) for a modern and authoritative feel. Background color: A dark, desaturated violet (#1D1822) to provide a sophisticated base. Accent color: A bright, electric blue (#4D7AFF) to highlight interactive elements and calls to action, providing dynamic contrast.
- Headlines: 'Space Grotesk' (sans-serif) for a contemporary, tech-inspired appearance. Body Text: 'Inter' (sans-serif) for clear readability and a professional, clean aesthetic.
- Use sleek, minimalist line icons with potential glowing effects to align with the futuristic AI theme. Icons should be clear and consistent across the platform for easy identification of studio types and actions.
- A structured and modular grid-based layout for dashboards and studio listings, ensuring content is organized and easily digestible. Implement responsive design principles to maintain optimal usability on both desktop and mobile interfaces.
- Incorporate subtle, smooth transitions for navigation elements, hover states on studio cards, and data loading or updates. Animations should be clean and contribute to a premium, high-tech user experience without being distracting.
